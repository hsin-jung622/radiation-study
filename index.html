法規 45分 18:30–19:15｜劑量是什麼＋劑量限度 ▶

▌先搞懂「劑量」三個名詞（考試最常混）

想像你的身體是一個靶，輻射打過來：

  • 吸收劑量（Gy）：身體實際吸收了多少能量。1 Gy = 每公斤組織吸收1焦耳能量。
  • 等價劑量（Sv）：把不同輻射種類的傷害程度考慮進去。α粒子傷害比X光大20倍，所以α要乘20。等價劑量 = 吸收劑量 × 加權因數。
  • 有效劑量（Sv）：再把「哪個器官被照到」考慮進去。性腺比皮膚重要，所以性腺的等價劑量要乘比較大的係數。
記憶方法

Gy → Sv的第一步是乘「輻射加權因數」（考慮輻射種類）；Sv → 有效劑量是再乘「組織加權因數」（考慮被照器官）。考試通常問有效劑量。

最常考的陷阱

「個人劑量 = 體外曝露 + 體內曝露」，不包含背景輻射和醫療曝露。選擇題常問「哪些要加總？」→ 只有B體外＋C體內。

▌四個重要名詞定義（問答題常考）

  • 確定性效應：有「門檻劑量」，超過才會發生，而且劑量越高越嚴重。例子：白內障、不孕症。
  • 機率性效應：沒有門檻，劑量越高發生的「機率」越大，但嚴重度不隨劑量變。例子：癌症。
  • 職業曝露：輻射工作人員因為工作受到的曝露（不含醫療和背景）。
  • 緊急曝露：事故中自願超過正常限度去救人。緊急曝露上限：有效劑量100 mSv（一次），救命時可達250 mSv。
物理 45分 19:15–20:00｜原子結構＋四種衰變＋半衰期公式 ▶

▌原子是什麼（30秒建立概念）

原子核在中間，裡面有質子（帶正電）和中子（不帶電）。外面繞著電子（帶負電）。
質子數叫「原子序Z」，質子＋中子叫「質量數A」。同一種元素的原子序相同，但中子數可以不同，這叫「同位素」。

▌四種衰變（背表格，每次考1–2題）

衰變丟出什麼Z變化A變化生活化記法
α衰變α粒子（氦核²₂He⁴）-2-4丟一個氦核出去，Z減2、A減4
β⁻衰變電子＋反微中子+1不變中子變質子，多出一個電子跑掉
β⁺衰變正子＋微中子-1不變質子變中子，多出一個正子跑掉
γ衰變γ光子不變不變激發態原子釋放能量，但不改變核的組成

▌半衰期公式（一定要會，計算題必用）

$$A(t) = A_0 \times \left(\frac{1}{2}\right)^{\Large t/T_{1/2}} = A_0 \cdot e^{-\lambda t}$$
$$\lambda = \frac{0.693}{T_{1/2}}$$

說人話：每過一個半衰期，活度剩一半。過2個T½剩¼，過3個剩⅛，過10個剩約0.1%

$$1\,\text{Ci} = 3.7\times10^{10}\,\text{Bq} \qquad 1\,\text{Bq} = 1\,\text{次/秒}$$
必記的五個核種半衰期（每年計算題都出現）

Co-60：5.26年 / Cs-137：30年 / I-131：8天 / Tc-99m：6小時 / Mo-99：66小時

物理 45分 19:15–20:00｜X光與物質作用＋偵測器比較 ▶

▌X光與物質五種作用（每次考1–3題）

想像X光光子是一顆球打進物質裡，根據能量大小會發生不同事情：

作用能量範圍發生什麼考試重點
光電效應低能（<0.1 MeV）光子被完全吸收，打出一個電子μ ∝ Z³/E³（原子序越大、能量越低越容易）
康普吞散射中能（0.1–10 MeV）光子部分散射＋打出電子最重要！有計算公式，能量峰位必考
電子對生成>1.022 MeV光子消失，生成e⁺e⁻一對門檻能量1.022 MeV（兩個電子靜止質量）
相干散射極低能光子方向改變，能量不變能量不損失，選擇題用來排除
光核反應>8 MeV打出中子高能加速器才有，考試提到時選「>8 MeV」

▌康普吞能量計算（問答題必考）

第一步：算 α
$$\alpha = \frac{E_\gamma}{511\,\text{keV}}$$
第二步：回散射峰（光子180°散射後剩餘能量）
$$E_{\text{回散}} = \frac{E_\gamma}{1+2\alpha}$$
第三步：康普吞邊緣（電子獲得的最大動能）
$$E_{\text{邊緣}} = E_\gamma - E_{\text{回散}}$$
全能峰 = 原始光子能量，不需計算
$$E_{\text{全能峰}} = E_\gamma$$

Cs-137（662 keV）範例：
α = 662/511 = 1.296　→　E回 = 662/(1+2×1.296) = 184 keV　→　E邊 = 662−184 = 478 keV　→　全能峰 = 662 keV

為什麼要背這個

NaI(Tl)偵測器量Cs-137的題目幾乎每年都出，問「全能峰、康普吞邊緣、回散射峰各在哪個能量」。把上面這個計算練熟，看到Cs-137就能秒算。

▌第2類：永久停用的情形與後續處理（出現頻率：高，113、114都出）

「何種情形被視為永久停用？永久停用後設施經營者應如何處理？」

視為永久停用的情形（答這3點）

  • 安全條件與原核准內容不符，設施經營者未依規定申請核准停止，持續達一年以上
  • 核准停止使用期間，主管機關認定有污染環境、危害人體健康且無法改善或已不堪使用
  • 主管機關廢止其許可證

永久停用後的處理（答這幾點）

  • 列冊陳報主管機關
  • 退回原製造或銷售者、轉讓、以放射性廢棄物處理
  • 以放射性廢棄物處理，處理期間不超過3個月

▌第3類：嚴重污染環境的標準（出現頻率：極高，幾乎每次都出）

「請說明嚴重污染環境的輻射標準」
  • (1) 一般人年有效劑量達10 mSv
  • (2) 一般人體外曝露劑量率：1小時內超過0.2 mSv
  • (3) 空氣中放射性核種濃度：2小時內平均濃度超過年連續空氣排放物濃度的1000倍
  • (4) 水中放射性核種濃度：2小時內平均濃度超過年連續水中排放物濃度的1000倍
  • (5) 土壤的嚴重污染面積：達1000平方公尺以上
注意：嚴重污染年有效劑量是10 mSv，不是5 mSv

有些舊題答案寫5 mSv（那是另一個法條的「擅自改變場所外放射性物質含量」的標準）。嚴重污染環境標準是10 mSv。近幾年答案已統一為10 mSv。

▌游離腔是什麼（30秒建立概念）

游離腔就是一個裝著空氣的小房間。輻射打進去，把空氣中的分子打成離子（帶電），離子被電極收集形成電流。電流大小告訴你輻射有多強。

▌Bragg-Gray理論計算步驟（每次問答題必考之一）

1
空氣質量
$$m = \rho_{\text{air}} \times V = 1.293\times10^{-3}\,\frac{\text{g}}{\text{cm}^3} \times V\,\text{cm}^3 \times 10^{-3}\,\frac{\text{kg}}{\text{g}}$$
2
每秒離子對數（e = 1.6×10⁻¹⁹ C）
$$n_{\text{ip}} = \frac{I}{e} = \frac{I}{1.6\times10^{-19}}$$
3
空氣吸收劑量率（W = 33.97 eV/ip）
$$\dot{D}_{\text{air}} = \frac{n_{\text{ip}} \times W \times 1.6\times10^{-19}}{m}\quad\text{(Gy/s)}$$
4
腔壁吸收劑量率（Bragg-Gray）
$$\dot{D}_{\text{wall}} = \dot{D}_{\text{air}} \times S_{\text{wall/air}}$$
記住這個固定流程，題目給的數字代入就能算

113年第1次物理問答第4題就是這題型，答案是20 mGy/h。只要步驟對，即使算錯數字也能拿部分分數。

物理計算 60分 19:00–20:00｜物理問答4大計算類型完整步驟 ▶

▌計算類型1：有效半衰期（出現頻率：極高）

「某核種物理半衰期15天，生物半衰期45天，初始活度6 mCi，30天後體內剩多少？」
步驟1：算有效半衰期
$$\frac{1}{T_{\text{eff}}} = \frac{1}{T_{\text{物}}} + \frac{1}{T_{\text{生}}} = \frac{1}{15}+\frac{1}{45} = \frac{4}{45} \quad\Rightarrow\quad T_{\text{eff}} = \frac{45}{4} = 11.25\text{ 天}$$
步驟2：代入半衰期公式
$$A(30) = 6\,\text{mCi}\times\left(\frac{1}{2}\right)^{30/11.25} = 6\times e^{-1.848} = 6\times 0.158 \approx 0.95\,\text{mCi}$$
答：30天後體內剩約 0.95 mCi

▌計算類型2：NaI能譜峰位（出現頻率：極高，幾乎每年）

「以NaI(Tl)量Cs-137的662 keV，計算全能峰、康普吞邊緣、回散射峰」
步驟1：算 α
$$\alpha = \frac{E_\gamma}{511} = \frac{662}{511} = 1.296$$
步驟2：算回散射峰
$$E_{\text{回}} = \frac{662}{1+2\times1.296} = \frac{662}{3.592} = 184\,\text{keV}$$
步驟3：算康普吞邊緣
$$E_{\text{邊}} = 662 - 184 = 478\,\text{keV}$$
步驟4：全能峰
$$E_{\text{全能峰}} = E_\gamma = 662\,\text{keV}$$
答：全能峰662 keV、康普吞邊緣478 keV、回散射峰184 keV

▌計算類型3：GM計數效率（出現頻率：高）

「5年前買0.25 μCi的Co-60，GM計數器測到2×10⁴ cpm，無感時間300 μs，求：(1)損失比率 (2)計數效率」
步驟1：算現在活度
$$A = A_0 \cdot e^{-\lambda t} = 0.25\,\mu\text{Ci}\times3.7\times10^4\,\frac{\text{Bq}}{\mu\text{Ci}}\times e^{-\frac{0.693\times5}{5.26}} = 4787\,\text{Bq}$$
步驟2：死時間修正（τ = 300 μs = 300×10⁻⁶ s）
$$R_{\text{true}} = \frac{R_0}{1 - R_0\cdot\tau} = \frac{20000/60}{1-\frac{20000}{60}\times300\times10^{-6}} = 370\,\text{cps} = 22222\,\text{cpm}$$
步驟3：損失比率
$$\text{損失比率} = 1 - \frac{R_0}{R_{\text{true}}} = 1 - \frac{20000}{22222} = 10\%$$
步驟4：計數效率（用觀測到的 cpm ÷ 真實活度 dpm）
$$\varepsilon = \frac{R_0\,(\text{cpm})}{A\,(\text{Bq})\times60} = \frac{20000}{4787\times60} = 7\%$$
答：(1)損失比率10%；(2)計數效率7%

▌計算類型4：平均壽命證明（出現頻率：高，幾乎每兩年一次）

「請證明放射性同位素的平均壽命為1/λ」
$$\tau = \frac{1}{N_0}\int_0^{\infty} t\left|\frac{dN}{dt}\right|dt = \frac{1}{N_0}\int_0^{\infty} t\cdot\lambda N_0 e^{-\lambda t}\,dt = \lambda\int_0^{\infty} t\,e^{-\lambda t}\,dt$$
用分部積分：∫₀^∞ t·e^(−λt) dt = 1/λ²
$$\tau = \lambda \times \frac{1}{\lambda^2} = \boxed{\dfrac{1}{\lambda}} \quad\blacksquare$$
結論：平均壽命 τ = 1/λ = T½ / 0.693 = 1.443 × T½

法規多拿8分的方法

最省力問答題每題多寫1–2個關鍵詞 → +12分▶

法規問答按關鍵詞給分，每個詞約2分。現在估計每題5分，代表漏了2–3個詞。補救方式很簡單：把標準答案的詞一個一個抄下來，考試時照單全開寫。

物理多拿10分的方法

+7分串聯衰變「畫圖說明」不算數字也有7分▶

串聯衰變題通常分3小問：(1)畫圖3–4分 (2)說明趨勢2–3分 (3)算數字3–4分。就算第(3)點算錯，前兩點還是有5–7分。

要畫/寫的內容得分

侵入途徑（3條）對應防護方法

②經由外傷傷口侵入若有外傷避免在污染區工作；必須時將傷口妥善包紮
③經由無傷口的皮膚吸收穿著工作服或防護衣，防止皮膚接觸
+3–5分/題計算題「步驟部分給分」策略▶

算錯也能拿5–7分，前提是步驟要寫對。物理問答每個小問獨立給分。

  • 先寫公式名稱和公式：就算不會算，寫出公式就有1–2分
  • 把已知數全部列出來：I = ?、V = ?、W = ?…整理已知本身就有分
  • 算錯了但繼續往下算：用錯的中間值繼續推導，步驟邏輯對還是有分
  • 每步驟都寫單位：Gy/s、Bq、cpm…顯示你知道在算什麼
